The Careers Guidance App

Overview

The Careers Guidance App is an iOS mobile application that I developed for my Final Year Project. The app is written in Swift using the SwiftUI and UIKit Frameworks in XCode.

The Careers Guidance App aids users seeking third-level undergraduate courses in Ireland by offering a course-matching questionnaire. This questionnaire collects data on personal and academic interests to generate course suggestions aligned with the user's traits and skills. These recommendations are formulated based on the user's questionnaire responses, enhancing their course search experience.

Key Features / Functionality
  • Option to log in as an existing user, register as a new user or continue as a guest
  • Course Matching Questionnaire
  • Option to save questionnaire results to the Results Dashboard to view again at a later time
  • In the Results Dashboard, users can view an analysis of their results (This feature is not available to guests)

Event Journal App

Overview

Memory Lane Event Journal is an Android mobile application I developed for my Mobile Application Development module in my Masters degree. I developed this app using Kotlin and Android Studio.

This app acts as a digital journal where users can document their events / experiences (e.g. birthdays, holidays, parties) to look back on.

Key Features / Functionality:
  • User authentication via email/password or Google Authentication
  • Add, view, edit, and delete events
  • Input event details like names, types, dates, times, descriptions, budgets, and images
  • Search events by name
  • Favorite events
  • Toggle/filter between all, favorite, and past events
  • Update profile picture
  • Integrated maps for event locations, filterable by favourited events and/or past event

YouTuber App

Overview

The YouTuber App is an Android mobile application I developed for my Mobile Application Development 1 module in my final year of my undergraduate degree. I also developed this app using Kotlin and Android Studio.

The YouTuber App offers a way for users to store information about their favourite YouTubers such as their name, channel name, date of birth, an image of them, their work location via Google Maps and a rating.

Key Features / Functionality
  • Option to log in as an existing user or register as a new user
  • Add YouTubers with information including their name, channel name, date of birth, image, their work location (via google maps API) and a rating
  • List YouTubers
  • Edit YouTubers
  • Delete YouTubers
  • Search and filter YouTubers
  • Mark YouTubers as favourite
