Tutors Card Layout Enhancement

As part of my volunteer work on Tutors, an open-source learning platform, I redesigned the course card layout to enhance user engagement and accessibility. I focused on making the course cards easier to navigate, ensuring a more intuitive experience for users.

The embedded presentation outlines these changes and rationale behind them, which I created to propose the changes to the founder of Tutors. I used Svelte and Tailwind CSS to implement these changes.

Tutors Card Layout Enhancement

As part of my work on Tutors, I enhanced the search button and filtering functionality to improve accessibility. I focused on providing easier exit options for users to close the search and filter menus without losing their progress. Additionally, I ensured the filtering options were visible and accessible on smaller screens, providing a more intuitive experience for users working with smaller windows or split-screen views.

The embedded presentation outlines these changes and rationale behind them. I used Svelte and Tailwind CSS to implement these changes. These changes will be merged in the upcoming rebuild of Tutors in December 2024.

Memory Lane Event Planning Design

As part of my User Experience Design Module in my Master's coursework, I invented a fictional company called Memory Lane Event Planning, that specialises in upscale event planning. The goal of Memory Lane Event Planning, is to craft unforgettable experiences for various occasions like weddings, milestone birthdays, and anniversaries.

The embedded document outlines the complete design process, including personas, scenarios, a sitemap, content strategy, competitor analysis, sketches, wireframes, and prototypes. It also provides detailed rationale behind each design decision to ensure an intuitive and effective user experience.

There is also the usability testing document for this project. This hypothetical usability testing document outlines the setup for evaluating the home page of Memory Lane Event Planning's website. The goal is to assess the usability and functionality by observing user interactions and identifying potential strengths and weaknesses in the design, features, and navigation. It covers aspects such as participant recruitment, test context, and the evaluation methods used to refine the overall user experience.
